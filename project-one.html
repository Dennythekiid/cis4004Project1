Hybrid Boggle/Scrabble desktop game implemented in Java. Designed and built an event-driven UI and core word-search logic; focused on clean UX and efficient algorithms.

Description

Developed a Java desktop game that blends Boggle mechanics with Scrabble-style scoring. Implemented the back-end word search using depth-first search with pruning and dictionary lookup; structured the code with OOP principles to separate game logic from the UI. Built an intuitive interface that guides users through gameplay and scoring.

  • Algorithms (DFS/backtracking)
  • Data structures (tries/sets)
  • Unit testing (JUnit)
  • MVC separation of concerns
